# Wasm Frontend

This crate is for createing the frontend of our application
it is a wasm binding

# To run
Install `wasm-pack` and `yarn`
```
cargo install wasm-pack
```
```
npm i -g yarn
```

run on linux and mac
```
./build-dev.sh
```

do 
```
./please-translate-build-dev.sh-to-powershell.bat
```
